## 增长飞轮

```
老板发招聘（免费）
  → 求职者涌入注册
    → 求职者也有生意 → 转化付费
    → 求职者推荐给老板朋友

公众号推送"本周GST提醒"
  → 点击进入Web App → 注册

免费找GP → 用户留存 → 发现商业功能 → 付费

机票监控 → 高频使用 → 口碑传播
```
